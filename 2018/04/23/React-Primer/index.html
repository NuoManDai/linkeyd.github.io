State

React中state是在通过与用户的交互，实现不同状态，然后渲染 UI，让用户界面和数据保持一致。state只作用于单个组件内部，并且由内部机制提供实时渲染。它由state,setState();
state:用户初始化以及函数调用时使用.
setState:用与修改数据时使用。它的内部是由一个柯里化函数实现。

Props

React中props是在组件之间提供数据交互。由上层组件传输给下层组件，由于其单独的特性。所以该属性中的所有数据并不提供在组件内的单独交互，可以将属性赋值给state来进行内部的交互。
